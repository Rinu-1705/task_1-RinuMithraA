# Bonnie - A RuleBased ChatBot Assistant
**Bonnie** is a friendly, rule-based chatbot built with vanilla HTML, CSS, and JavaScript. It provides an interactive conversational experience with a clean, modern user interface. The chatbot responds to user inputs using a predefined set of rules and patterns, making it perfect for learning chatbot fundamentals or serving as a lightweight conversational AI assistant.

## Features

**Conversational Responses** 
- Multiple response categories with contextual replies
- Greetings and farewells
- Personality-driven interactions
- Humorous jokes and interesting facts
- Time/date awareness
- Trivia and knowledge sharing

**User-Friendly Controls**
- Send messages via button click or Enter key
- Auto-focus on input field
- Auto-scroll to latest messages
- Helpful command suggestions

### Suggested Commands

Try these commands:

- **Greetings**: `hello`, `hi`, `hey`, `good morning`
- **Questions**: `how are you`, `your name`, `what can you do`
- **Entertainment**: `tell me a joke`, `say something interesting`
- **Casual**: `thanks`, `goodbye`, `what's new`
- **Trivia**: `what is pi`, `what is 2+2`
